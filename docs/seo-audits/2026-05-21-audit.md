### 9.1 Constat majeur : pivot organique non documenté

[`STRATEGIE-EDITORIALE.md`](../../STRATEGIE-EDITORIALE.md) §3 prévoit 6 clusters (Audit IT, Stratégie IA PME, Secteurs Métiers, Compliance, MCP, Vente Commerciale).
La réalité du trafic = **un 7e cluster Claude Code / Odoo / Prompt engineering** qui draine **~90 % des clics** sur 90j.

Deux options :
- **(A) Capitaliser** : créer un mini-pilier `/claude-code-pour-pme` + 3-4 articles satellites + une **offre dédiée** (formation Claude Code dirigeants/CTO PME, 1/2 jour à 650 €). Risque : sortie du périmètre 78/95, audience tech non-cliente directe.
- **(B) Encadrer** : conserver les 2 articles comme "produits d'appel" mais investir le contenu prioritaire sur les clusters initiaux. Inconvénient : on laisse 100+ clics/mois sans funnel adapté.
