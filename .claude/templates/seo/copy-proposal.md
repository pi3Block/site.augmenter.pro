# Template — Propositions de copy A/B/C

Format standard pour les propositions de réécriture de copy livrées par `/seo-audit` (Phase 9.2). Pour chaque page clé, proposer **3 variations de ton** pour permettre un arbitrage rapide.

## Structure par page

```markdown
#### Page : <path>

**Hero H1 — version actuelle**
> <texte actuel>

**Proposition A (ton provocateur — douleur frontale)**
> <nouveau H1>
> <sous-titre>

**Proposition B (ton aspirationnel — bénéfice chiffré)**
> <nouveau H1>
> <sous-titre>

**Proposition C (ton pragmatique — méthode/preuve)**
> <nouveau H1>
> <sous-titre>

**Meta title (≤ 60 chars) — version actuelle** : ...
**Propositions** :
1. ...
2. ...
3. ...

**Meta description (≤ 155 chars) — version actuelle** : ...
**Propositions** :
1. ...
2. ...

**CTA principal — version actuelle** : « ... »
**Propositions** :
1. ...
2. ...
```

## Les 3 tons — grille de décision

| Variante | Promesse | Émotion primaire | Quand l'utiliser |
|----------|----------|------------------|------------------|
| **A — Provocateur** | Nomme la douleur | Inconfort reconnu | Page d'accroche, home hero, accroches publicitaires |
| **B — Aspirationnel** | Chiffre un bénéfice | Désir, projection | Pages commerciales, services premium |
| **C — Pragmatique** | Méthode + preuve | Confiance, sérieux | Pages méthodo, études de cas, pages techniques |

## Pages à couvrir systématiquement

- Home (hero + sections stratégiques — trust, services, pricing, CTA)
- `/prestations`
- `/approche`
- `/idees`
- `/blog` (index)
- `/contact`
- Top 5 articles par trafic GSC
- Pages service/secteur si existantes

## Exemples de patterns (à calibrer au cas par cas)

**Provocateur (A)**
- « ChatGPT dans votre PME : 80 % des dirigeants ratent les 3 chantiers rentables »
- « Votre dernier devis BTP a mis 4 jours à sortir. Voici pourquoi. »
- « L'IA ne va pas remplacer votre artisan. Le voisin qui s'en sert, si. »

**Aspirationnel (B)**
- « Dirigeant PME : 12 h libérées par semaine grâce à 3 automatisations IA »
- « De devis en 3 jours à devis en 30 minutes — la méthode testée sur 12 PME 78/95 »

**Pragmatique (C)**
- « Audit IA PME : la méthode en 5 étapes qui identifie vos chantiers rentables »
- « Consultant IA PME Yvelines — diagnostic 60 min, plan d'action chiffré »

## Contraintes

- **Mot « gratuit » interdit** — utiliser « offert », « sans engagement », « inclus »
- **H1 ≤ ~70 caractères visuels** (lisibilité mobile)
- **Meta title ≤ 60 chars** (sinon tronqué par Google)
- **Meta description ≤ 155 chars** (sinon tronquée)
- **Power words** : Gratuit interdit → substituer par « offert », « instant », « immédiat », « sans CB »
- **Géo-ciblage** : 78, 95, Yvelines, Val d'Oise, Saint-Germain, Versailles, Cergy — quand pertinent
- **Cohérence** : une fois la variante retenue, propager **à l'identique** sur toutes les occurrences du CTA/slogan (Phase 9.5 du seo-audit)
